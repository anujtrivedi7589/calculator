<h1 align="center">🧮 Calculate Me! – Web Calculator</h1>

<p align="center">
  A simple and efficient web calculator built using HTML, CSS, and JavaScript.
</p>

<hr>

<h2>🚀 Overview</h2>
<p>
<b>Calculate Me!</b> is a lightweight calculator web app that performs basic arithmetic operations 
with a clean and user-friendly interface.
</p>

<h2>✨ Features</h2>
<ul>
  <li>➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division</li>
  <li>🔢 Interactive button-based input</li>
  <li>⚡ Instant results using JavaScript</li>
  <li>🧹 Clear button to reset input</li>
  <li>🎯 Simple and minimal UI</li>
</ul>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li><b>HTML5</b></li>
  <li><b>CSS3</b></li>
  <li><b>JavaScript</b></li>
</ul>

<h2>🧩 How It Works</h2>
<ol>
  <li>Click buttons to enter numbers and operators</li>
  <li>Input appears on the display</li>
  <li>Press <b>=</b> to calculate</li>
  <li>Press <b>C</b> to clear</li>
</ol>

<h2>📂 Project Structure</h2>
<pre>
index.html   → Main UI  
style.css    → Styling  
utils.css    → Utility classes  
script.js    → Logic  
</pre>
